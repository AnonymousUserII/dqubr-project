# dQubr Project
Some random application for cubers. Made in Python because.

## Required Dependencies
- PyGame

## Current Features
- Interactive virtual cube + indicators

## Controls
Cube turning is processed in SiGN notation.

|  Feature | Key(s) |
|---------:|:-------|
|     Undo | `      |
|    Reset | F5     |
| Scramble | F8     |
